Um clearfix é uma maneira de um elemento limpar ou corrigir automaticamente seus elementos, de forma que não seja necessário adicionar marcações adicionais. É geralmente usado em layout flutuante, onde os elementos flutuam para serem empilhados horizontalmente.

Este exemplo usa consultas de mídia para reorganizar as imagens em diferentes tamanhos de tela: para telas maiores que 700px de largura, ele mostrará quatro imagens lado a lado, para telas menores que 700px, mostrará duas imagens lado a lado. Para telas menores que 500px, as imagens serão empilhadas verticalmente (100%).
